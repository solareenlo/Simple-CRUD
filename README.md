# Simple-CRUD
React+Redux+Express+MongoDBで作るものすごくシンプルなCRUDアプリ

## Usage
```bash
# GitHubからソースコードをダウンロード
git clone https://github.com/solareenlo/Simple-CRUD.git

# ソースコードがあるディレクトリに移動
cd Simple-CRUD

# yarnを使ってパッケージをダウンロード
yarn

# 別ウィンドウでmongodbを起動
mongod

# yarnを使ってアプリ起動
yarn start
```
そして, 任意のブラウザで`http://localhost:3001`を開く.
